Temel Amaç

Elly, Spring Boot 3.5 üzerine inşa edilmiş headless bir CMS ve hızlı MVP platformudur. Sayfalar, bileşenler, dinamik formlar, asenkron e-posta ve kullanıcı kimlik doğrulaması — hepsi kiracı bazında izole. Ekipler saatlerce çalışarak tam bir web sitesi oluşturabilir.

Dört Temel
İçerik — Page → Component → Banner / Widget → Post → Comment / Rating
Auth — User → Role → Permission · Login · Register · RefreshToken · OAuth2
Form — FormDefinition (JSONB şema) → ConditionEvaluator → FormSubmission
E-posta — MailAccount (kiracı bazlı) → asenkron kuyruk → Thymeleaf → Gmail SMTP
Tüm Alan Varlıkları
Page Component Banner Widget Post Form User Permission Comment Rating Assets Email MailAccount RefreshToken SeoInfo Role FormDefinition FormSubmission
Hızlı MVP: Sayfaları dağıt, Bileşenler ekle, Post'ları yayınla — dakikalar içinde. Dinamik Form ve asenkron E-posta ile temel CMS yapısını bozmadan genişlet.
