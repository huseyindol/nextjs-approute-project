Overview
What is Elly?
Core Purpose

Elly is a headless CMS and rapid MVP platform built on Spring Boot 3.5. It manages pages, components, dynamic forms, async email, and user auth — all isolated per tenant. Teams go from zero to a full website in hours, not weeks.

Four Pillars
Content — Page → Component → Banner / Widget → Post → Comment / Rating
Auth — User → Role → Permission · Login · Register · RefreshToken · OAuth2
Forms — FormDefinition (JSONB schema) → ConditionEvaluator → FormSubmission
Email — MailAccount (per tenant) → async queue → Thymeleaf → Gmail SMTP
All Domain Entities
Page Component Banner Widget Post Form User Permission Comment Rating Assets Email MailAccount RefreshToken SeoInfo Role FormDefinition FormSubmission
Quick MVP: deploy Pages, attach Components, publish Posts in minutes. Extend with dynamic Forms and async Email without restructuring core CMS.
